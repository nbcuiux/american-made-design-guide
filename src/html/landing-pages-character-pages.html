Since American Made tells a character driven story, we wanted to provide a way for the users to connect with each of the key characters and subsequently become intrigued and drawn to the film.

Character Splash Pages

The film site singmovie.com features an interactive character exploration section with character bios.

The interaction invites the user to learn and invest in the characters. American Made, as we envision, is a character-driven film and featuring a similar interaction could help the user invest and build a relationship with the characters.
